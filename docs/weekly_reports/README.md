# 周报归档 (Weekly Reports)

本目录归档每周工作周报。

## 命名规则

```
YYYYMMDD.md
```

- 日期取**该周报截止日（周末）**，例如 `20260618.md` 表示 2026-06-12 ~ 2026-06-18 这一周。
- 按时间顺序排列，最新的在后面。

## 内容约定

每份周报建议包含：

1. **本周工作全景**：按类别（模型设计 / Bug 修复 / 训练 / 推理 / 评估）分表，附 commit hash。
2. **关键指标进展**：各数据集 × 各任务（extraction/completion）的 APLS/TOPO/IoU/F1 等指标对比。
3. **问题与排查**：本周遇到的 bug、根因、修复方式（便于回溯）。
4. **下周计划**：明确下一步要验证的方法 / 要跑的实验。
