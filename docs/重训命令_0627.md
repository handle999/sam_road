## 一、didi_xian

### 1.1 extraction 基线(复用 ckpt,只跑 infer+eval)

extraction 不碰 partial,用 0625 已训的 ckpt,只重新推理+评估(口径对齐)。

```bash
python run.py --task extraction --dataset didi_xian \
    --run-id extraction_didi_xian_20260625_015721 \
    --steps infer,eval --resume-run \
    --checkpoint epoch:9 --precision 32 --gpus 6
```
- `--resume-run`:复用 0625 的 run(infer 可能已完成,会跳过;若想强制重跑见注)
- `--checkpoint epoch:9`:用第10轮 ckpt
- 产物:`runs/extraction_didi_xian_20260625_015721/`
- 0625 结果:APLS=0.4288, TOPO F1=0.5889

> 注:若 infer 已完成被跳过,想重跑可用新 run-id:
> `--run-id extraction_didi_xian_rerun --steps infer,eval --checkpoint <0625 ckpt绝对路径> --precision 32 --gpus 6`

### 1.2 completion 重训(partial 变了,必须重训)

```bash
python run.py --task completion --dataset didi_xian \
    --checkpoint epoch:9 --precision 32 --epochs 10 --gpus 6
```
- 默认 steps=train,infer,eval(全流程)
- 训练用按边随机 partial(默认),traj 训练增强(didi_xian 有 traj)
- 产物:`runs/completion_didi_xian_<新时间戳>/`
- 0625 结果:APLS=0.5878, TOPO F1=0.7925(partial 旧版);新版预期相近或略好

### 1.3 completion 消融(用 1.2 的新 ckpt)

等 1.2 训完后,找到新 ckpt 路径:
```bash
ls runs/completion_didi_xian_<新时间戳>/train/checkpoints/*epoch=09*
# 记下路径, 下面用 $CKPT_DIDI 替代
```

**目标1:无 traj 无 rn(退化 ≈ extraction)**
```bash
python run.py --task completion --dataset didi_xian \
    --run-id completion_didi_xian_notraj_norn_ep9 \
    --steps infer,eval \
    --checkpoint $CKPT_DIDI \
    --no-traj --no-input-graph --precision 32 --gpus 3
```
- `--no-traj --no-input-graph`:关掉 traj 和 rn,纯 extraction 退化
- 0625 结果:APLS=0.4332, TOPO F1=0.6081

**目标2:仅 traj(无 rn)** *(偏离主设定,traj 应训练用,保留参考)*
```bash
python run.py --task completion --dataset didi_xian \
    --run-id completion_didi_xian_trajonly_ep9 \
    --steps infer,eval \
    --checkpoint $CKPT_DIDI \
    --no-input-graph --precision 32 --gpus 3
```
- 只关 rn,带 traj(不加 --no-traj)
- 0625 结果:APLS=0.4460, TOPO F1=0.6207

**目标2+:仅 rn(无 traj)** *(最接近主设定:traj 训练不用,推理 img+rn)*
```bash
python run.py --task completion --dataset didi_xian \
    --run-id completion_didi_xian_rnonly_ep9 \
    --steps infer,eval \
    --checkpoint $CKPT_DIDI \
    --no-traj --precision 32 --gpus 3
```
- 只关 traj,带 rn
- 0625 结果:APLS=0.5709, TOPO F1=0.7728
